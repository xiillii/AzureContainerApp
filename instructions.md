# POC Azure Container Apps

# Requerimientos

1. Un sistema de microservicios, que se pueda desplegar en un ambiente de Azure Container Apps.
2. Dos frontends en Dot NET 8.x. Con acceso desde internet.
3. Dos backends en Dot NET 8.x, para cada frontend.
4. Una base de datos SQL en Azure.
5. Un sistema de autenticación y autorización.
6. Un sistema de almacenamiento para leer y escribir archivos. Azure Storage Account, ya sea Blob Storage o File Storage.
7. Un sistema de monitoreo y logging.
8. El codigo colocarlo en el directorio ./srcs

La POC debe ser simple, quizas un TODO app, pero debe cumplir con todos los requerimientos mencionados.
