# POC Azure Container Apps

# Requerimientos

1. Un sistema de microservicios, que se pueda desplegar en un ambiente de Azure Container Apps.
2. Dos frontends en Dot NET 8.x
2. Dos backends en Dot NET 8.x, para cada frontend.
3. Una base de datos SQL en Azure.
4. Un sistema de autenticación y autorización.
5. Un sistema de almacenamiento para leer y escribir archivos. Azure Storage Account, ya sea Blob Storage o File Storage.
6. Un sistema de monitoreo y logging.
7. El codigo colocarlo en el directorio ./srcs
